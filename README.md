# FastAPI ToDo API

A production-style backend API built with FastAPI.

Features:
- JWT Authentication
- Async SQLAlchemy
- User-owned todos
- Offset Pagination
- Cursor Pagination
- Dockerized deployment

Tech Stack:
- FastAPI
- SQLAlchemy (Async)
- SQLite
- Docker

Endpoints:
- /auth/register
- /auth/login
- /todos
